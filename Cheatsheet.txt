<html></html>
<head></head> - Metadata
	<title></title>
	<style> - style information
	<base> - default URL
	<link> - link to external stylesheet (<link rel="stylesheet" type="text/css" href="styles.css")
	<meta> - metadata (charset, name, content)
	<script> - embed client side script(javascript)
	<noscript> - message if scripts are not supported

	<footer></footer> - not required (copyright = "&copy; year, etc")

	<p></p> - paragraphs
	<div></div> - divide tag stand alone content
	<ol></ol> - ordered list "numbered list"
	<ul></ul> - unordered list "bullet points"
	<li></li> - list item
	<dl></dl> - definition list
	<dt></dt> - definition title "word defined"
	<dd></dd> - definition
	<address></address> - addresses - contact info
	<article></article> - articles, content that can be re-used elsewhere

		<tr></tr> - table row
			<td></td> - data in row, "<td> info</td><td> info 2</td>" - creats new column in the row

	<nav></nav> - group of links
	<time></time> - time tag
